命令行方式传文件：
1.在GitHub上创建了一个存储库，或者由您想要贡献的其他人拥有一个现有的存储库
2.在计算机上本地克隆存储库
（1）在GitHub上，导航到存储库的主页面。
（2）在存储库名称下，单击“ 克隆”或“下载”。
（3）在“使用HTTP克隆”部分中，单击  复制存储库的克隆URL。
（4）打开Git Bash
（5）将当前工作目录更改为您希望克隆目录的位置
（6）键入git clone，然后粘贴您在步骤2中复制的URL
（7）按Enter键。将创建您的本地克隆
3.在计算机上，将要上载到GitHub的文件移动到克隆存储库时创建的本地目录中
4.打开Git Bash
5.将当前工作目录更改为本地存储库
6.将文件暂存以提交到本地存储库 
   git add “文件名”
7.提交您在本地存储库中暂存的文件
   git commit -m“添加现有文件”
8.将本地存储库中的更改推送到GitHub 
   git push origin master
